# NeuroVideoAI - 智能運動姿態紀錄系統

一個基於 MediaPipe 的智能運動姿態分析與紀錄系統，專為醫療復健場景設計。

## 功能特色

- 📹 **即時姿態檢測**：使用 MediaPipe Pose 進行即時人體姿態檢測
- 🎥 **雙視角錄影**：支援指導影片與病人攝影機同步顯示
- 💾 **本機儲存**：所有資料儲存在瀏覽器本機，保護隱私
- 📱 **行動優化**：專為手機豎屏設計，提供專業商用軟體體驗
- 🏥 **醫療導向**：整合病人資訊管理與評估記錄

## 技術架構

- **前端框架**：原生 HTML/CSS/JavaScript
- **姿態檢測**：MediaPipe Pose
- **資料儲存**：IndexedDB + LocalStorage
- **響應式設計**：支援手機、平板、桌面多種裝置

## 快速開始

### 本地開發

1. 啟動本地伺服器：

```bash
cd demo
python3 -m http.server 8000
```

2. 在瀏覽器中開啟 `http://localhost:8000`

### 使用說明

1. **錄影模式**：
   - 選擇指導影片（可選）
   - 開啟攝影機
   - 輸入病人身分證字號
   - 選擇今日狀況評估
   - 開始錄影

2. **管理模式**：
   - 查看所有紀錄
   - 播放已錄製的影片
   - 刪除不需要的紀錄

## 瀏覽器支援

- Chrome/Edge (推薦)
- Safari
- Firefox

**注意**：需要支援 MediaRecorder API 和 IndexedDB 的現代瀏覽器。

## 隱私與安全

- 所有資料僅儲存在使用者裝置本機
- 不會上傳任何資料到外部伺服器
- 完全離線運作（除了載入 MediaPipe CDN）
